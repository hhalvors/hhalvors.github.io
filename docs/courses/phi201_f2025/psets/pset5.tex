Resources: Chapter 6 (pp 99-113) of \emph{How Logic Works}.

Prove the following sequents using the propositional logic rules plus
UE, UI, EI, and EE.

\begin{enumerate}
\item $\exists x(Fx\to Gx)\:\vdash\:\forall xFx\to\exists xGx$
\item $\forall x\exists y(Fy\to Gx)\:\vdash\:\forall yFy\to\forall xGx$
\item $\exists x(Fx\vee Gx),\neg \exists xFx\:\vdash\:\exists yGy$
\item $P\to \exists xFx\:\vdash\: \exists x(P\to Fx)$
\end{enumerate}
